# GS1 & Mã vạch

::: warning Trang đang được xây dựng
Nội dung trang này đang được soạn thảo. Vui lòng quay lại sau.
:::

Trang này sẽ hướng dẫn về chuẩn GS1 và ứng dụng mã vạch trong truy xuất nguồn gốc:

- Giới thiệu về tổ chức GS1
- Cấu trúc mã GTIN, GLN, SSCC
- Mã vạch 1D: EAN-13, Code 128
- Mã 2D: QR Code, Data Matrix
- Tích hợp GS1 vào hệ thống TXNG
