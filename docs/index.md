## 🚀 Bắt đầu trong 5 phút

Cài đặt và chạy hệ thống TXNG chỉ với vài bước đơn giản:

```bash
# Cài đặt CLI
npm install -g @txng/cli

# Khởi tạo dự án
txng init my-project

# Chạy môi trường dev
cd my-project && txng dev
```
